# 设置页面节点命名指南

## 整体结构

```
SettingsDialog (ConfirmationDialog)
├── MainContainer (VBoxContainer - 主垂直布局容器)
│   ├── TitleContainer (HBoxContainer - 标题水平布局)
│   │   └── TitleLabel (Label - 设置标题)
│   ├── BrightnessSection (VBoxContainer - 亮度设置区域)
│   │   ├── BrightnessHeader (HBoxContainer - 亮度头部布局)
│   │   │   ├── BrightnessLabel (Label - 亮度标签)
│   │   │   └── BrightnessSlider (HSlider - 亮度滑块)
│   │   └── BrightnessValueLabel (Label - 亮度值显示)
│   ├── SoundSection (VBoxContainer - 音效设置区域)
│   │   ├── SoundToggleRow (HBoxContainer - 音效开关行)
│   │   │   ├── SoundToggleLabel (Label - 音效开关标签)
│   │   │   └── SoundToggleButton (CheckButton - 音效开关按钮)
│   │   └── SoundVolumeRow (HBoxContainer - 音效音量行)
│   │       ├── SoundVolumeLabel (Label - 音效音量标签)
│   │       └── SoundVolumeSlider (HSlider - 音效音量滑块)
│   ├── BgmSection (VBoxContainer - BGM设置区域)
│   │   ├── BgmToggleRow (HBoxContainer - BGM开关行)
│   │   │   ├── BgmToggleLabel (Label - BGM开关标签)
│   │   │   └── BgmToggleButton (CheckButton - BGM开关按钮)
│   │   └── BgmVolumeRow (HBoxContainer - BGM音量行)
│   │       ├── BgmVolumeLabel (Label - BGM音量标签)
│   │       └── BgmVolumeSlider (HSlider - BGM音量滑块)
│   ├── SystemSection (VBoxContainer - 系统设置区域)
│   │   ├── SystemVolumeRow (HBoxContainer - 系统音量行)
│   │   │   ├── SystemVolumeLabel (Label - 系统音量标签)
│   │   │   └── SystemVolumeSlider (HSlider - 系统音量滑块)
│   │   └── FullscreenRow (HBoxContainer - 全屏模式行)
│   │       ├── FullscreenLabel (Label - 全屏模式标签)
│   │       └── FullscreenToggleButton (CheckButton - 全屏模式开关)
│   └── ButtonContainer (HBoxContainer - 按钮水平布局)
│       ├── CancelButton (Button - 取消按钮)
│       └── SaveButton (Button - 保存按钮)
```

## 命名规则说明

### 1. 容器命名规则

| 类型 | 命名格式 | 示例 |
|------|---------|------|
| 主容器 | XXXContainer | MainContainer, ButtonContainer |
| 区域容器 | XXXSection | BrightnessSection, SoundSection |
| 行容器 | XXXRow / XXXHeader | BrightnessHeader, SoundToggleRow |

### 2. 控件命名规则

| 类型 | 命名格式 | 示例 |
|------|---------|------|
| 标签 | XXXLabel | TitleLabel, BrightnessLabel |
| 滑块 | XXXSlider | BrightnessSlider, SoundVolumeSlider |
| 开关按钮 | XXXToggleButton | SoundToggleButton, BgmToggleButton |
| 普通按钮 | XXXButton | CancelButton, SaveButton |

## 创建步骤（带命名）

### 步骤 1: 创建根节点
1. 创建场景，根节点选择 **ConfirmationDialog**
2. 重命名为 **SettingsDialog**

### 步骤 2: 创建主容器
1. 在 SettingsDialog 下添加 **VBoxContainer**
2. 重命名为 **MainContainer**

### 步骤 3: 创建标题部分
1. 在 MainContainer 下添加 **HBoxContainer**
2. 重命名为 **TitleContainer**
3. 在 TitleContainer 下添加 **Label**
4. 重命名为 **TitleLabel**
5. 设置 TitleLabel 的文本为 "设置"

### 步骤 4: 创建亮度设置区域
1. 在 MainContainer 下添加 **VBoxContainer**
2. 重命名为 **BrightnessSection**
3. 在 BrightnessSection 下添加 **HBoxContainer**
4. 重命名为 **BrightnessHeader**
5. 在 BrightnessHeader 下添加 **Label**，重命名为 **BrightnessLabel**，文本设为 "亮度"
6. 在 BrightnessHeader 下添加 **HSlider**，重命名为 **BrightnessSlider**
7. 在 BrightnessSection 下添加 **Label**，重命名为 **BrightnessValueLabel**，用于显示数值

### 步骤 5: 创建音效设置区域
1. 在 MainContainer 下添加 **VBoxContainer**
2. 重命名为 **SoundSection**
3. 在 SoundSection 下添加 **HBoxContainer**，重命名为 **SoundToggleRow**
4. 在 SoundToggleRow 下添加 **Label**，重命名为 **SoundToggleLabel**，文本设为 "音效"
5. 在 SoundToggleRow 下添加 **CheckButton**，重命名为 **SoundToggleButton**
6. 在 SoundSection 下添加 **HBoxContainer**，重命名为 **SoundVolumeRow**
7. 在 SoundVolumeRow 下添加 **Label**，重命名为 **SoundVolumeLabel**，文本设为 "音效音量"
8. 在 SoundVolumeRow 下添加 **HSlider**，重命名为 **SoundVolumeSlider**

### 步骤 6: 创建BGM设置区域
1. 在 MainContainer 下添加 **VBoxContainer**
2. 重命名为 **BgmSection**
3. 在 BgmSection 下添加 **HBoxContainer**，重命名为 **BgmToggleRow**
4. 在 BgmToggleRow 下添加 **Label**，重命名为 **BgmToggleLabel**，文本设为 "BGM"
5. 在 BgmToggleRow 下添加 **CheckButton**，重命名为 **BgmToggleButton**
6. 在 BgmSection 下添加 **HBoxContainer**，重命名为 **BgmVolumeRow**
7. 在 BgmVolumeRow 下添加 **Label**，重命名为 **BgmVolumeLabel**，文本设为 "BGM音量"
8. 在 BgmVolumeRow 下添加 **HSlider**，重命名为 **BgmVolumeSlider**

### 步骤 7: 创建系统设置区域
1. 在 MainContainer 下添加 **VBoxContainer**
2. 重命名为 **SystemSection**
3. 在 SystemSection 下添加 **HBoxContainer**，重命名为 **SystemVolumeRow**
4. 在 SystemVolumeRow 下添加 **Label**，重命名为 **SystemVolumeLabel**，文本设为 "系统音量"
5. 在 SystemVolumeRow 下添加 **HSlider**，重命名为 **SystemVolumeSlider**
6. 在 SystemSection 下添加 **HBoxContainer**，重命名为 **FullscreenRow**
7. 在 FullscreenRow 下添加 **Label**，重命名为 **FullscreenLabel**，文本设为 "全屏模式"
8. 在 FullscreenRow 下添加 **CheckButton**，重命名为 **FullscreenToggleButton**

### 步骤 8: 创建按钮部分
1. 在 MainContainer 下添加 **HBoxContainer**
2. 重命名为 **ButtonContainer**
3. 在 ButtonContainer 下添加 **Button**，重命名为 **CancelButton**，文本设为 "取消"
4. 在 ButtonContainer 下添加 **Button**，重命名为 **SaveButton**，文本设为 "保存"

## 节点路径参考

在脚本中引用这些节点时，路径如下：

```gdscript
# 亮度相关
@onready var brightness_slider = $MainContainer/BrightnessSection/BrightnessHeader/BrightnessSlider
@onready var brightness_value_label = $MainContainer/BrightnessSection/BrightnessValueLabel

# 音效相关
@onready var sound_toggle_button = $MainContainer/SoundSection/SoundToggleRow/SoundToggleButton
@onready var sound_volume_slider = $MainContainer/SoundSection/SoundVolumeRow/SoundVolumeSlider

# BGM相关
@onready var bgm_toggle_button = $MainContainer/BgmSection/BgmToggleRow/BgmToggleButton
@onready var bgm_volume_slider = $MainContainer/BgmSection/BgmVolumeRow/BgmVolumeSlider

# 系统相关
@onready var system_volume_slider = $MainContainer/SystemSection/SystemVolumeRow/SystemVolumeSlider
@onready var fullscreen_toggle_button = $MainContainer/SystemSection/FullscreenRow/FullscreenToggleButton

# 按钮
@onready var cancel_button = $MainContainer/ButtonContainer/CancelButton
@onready var save_button = $MainContainer/ButtonContainer/SaveButton
```

## 命名技巧

1. **Section**: 用于表示一个完整的设置区域（包含多个相关控件）
2. **Row**: 用于表示一行控件
