# 海底无明日 - 代码解析文档

## 项目结构

本项目是一个基于 Godot 引擎的 2D 游戏，名为"海底无明日"。以下是项目的主要结构：

```
├── scenes/          # 场景文件
│   ├── adventure/   # 冒险场景
│   ├── base/        # 基地场景
│   ├── title/       # 标题场景
│   └── ui/          # UI 相关场景
├── scripts/         # 脚本文件
├── art/             # 美术资源
└── .godot/          # Godot 引擎配置
```

### 1. GameManager.gd

```gdscript
extends Node  # 继承自 Node 类

# 定义常量，存储场景路径

# 跳转到基地场景的函数
func goto_base() -> void:
    in_adventure = false  # 设置为不在冒险中
    get_tree().paused = false  # 确保游戏树不暂停
    get_tree().change_scene_to_file(BASE_SCENE)  # 切换到基地场景

# 开始冒险的函数
func start_adventure() -> void:
    in_adventure = true  # 设置为在冒险中
    get_tree().paused = false  # 确保游戏树不暂停
    get_tree().change_scene_to_file(ADVENTURE_SCENE)  # 切换到冒险场景

# 保存并退出的函数
func save_and_exit() -> void:
    # 先做占位，后面再接真正存档逻辑
    goto_title()  # 调用 goto_title 函数返回标题场景
```

### 3. AdventureScene.gd

```gdscript
extends Node2D  # 继承自 Node2D 类

@onready var pause_menu = $PauseMenu  # 当节点准备好时，获取 PauseMenu 子节点的引用

# 当节点准备好时调用的函数
func _ready() -> void:
    print("AdventureScene ready")  # 打印调试信息，表明冒险场景已准备就绪

# 处理未处理的输入事件
func _unhandled_input(event: InputEvent) -> void:
    if event.is_action_pressed("ui_cancel"):  # 如果按下了取消按钮（通常是 Esc 键）
        if pause_menu.visible:  # 如果暂停菜单可见
            pause_menu.close_menu()  # 调用 close_menu 方法关闭菜单
        else:  # 如果暂停菜单不可见
            pause_menu.open_menu()  # 调用 open_menu 方法打开菜单
```

### 4. BaseScene.gd

```gdscript
extends Node2D  # 继承自 Node2D 类

    click_tip.text = ""  # 清空点击提示的文本
```

## 场景结构

### 1. 标题场景 (TitleScreen.tscn)
- 包含游戏标题和开始游戏的提示
- 点击屏幕任意位置会跳转到基地场景

### 2. 基地场景 (BaseScene.tscn)
- 包含提示标签和点击提示
- 可以显示 NPC 对话
- 可以进入冒险场景

### 3. 冒险场景 (AdventureScene.tscn)
- 包含暂停菜单
- 按 Esc 键可以打开/关闭暂停菜单

## 游戏流程

1. **启动游戏** → 显示标题场景
2. **点击开始** → 跳转到基地场景
3. **在基地场景** → 可以与 NPC 对话，或者进入冒险场景
4. **在冒险场景** → 可以通过按 Esc 键打开暂停菜单
5. **暂停菜单** → 可以返回基地、返回标题或保存退出
