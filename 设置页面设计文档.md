# 游戏设置页面设计文档

## 概述

本文档详细说明如何在Godot引擎中创建游戏设置页面，包括UI组件的布局、层次关系和实现方法。设置页面将包含亮度、音效开关、系统音量、BGM音量等设置项。

## 设置页面触发流程

1. 玩家在冒险页面按ESC键打开暂停菜单
2. 点击暂停菜单中的"设置"按钮
3. 弹出设置页面
4. 在设置页面中调整各项设置
5. 点击"保存"或"取消"按钮关闭设置页面

## 详细实现步骤

### 1. 创建设置对话框场景

1. 在`scenes/ui/`目录下创建一个新的场景文件，命名为`SettingsDialog.tscn`
2. 选择`ConfirmationDialog`作为根节点，重命名为`SettingsDialog`
3. 在`SettingsDialog`节点下添加一个`VBoxContainer`节点，用于垂直布局所有设置项

### 2. 添加设置项组件

#### 2.1 标题部分

1. 在`VBoxContainer`下添加一个`HBoxContainer`节点，用于水平布局标题
2. 在`HBoxContainer`下添加一个`Label`节点，设置文本为"设置"，调整字体大小和样式

#### 2.2 亮度设置

1. 在`VBoxContainer`下添加一个`VBoxContainer`节点，用于垂直布局亮度设置相关组件
2. 在该`VBoxContainer`下添加一个`HBoxContainer`节点
3. 在`HBoxContainer`下添加一个`Label`节点，设置文本为"亮度"
4. 在`HBoxContainer`下添加一个`HSlider`节点，设置最小值为0，最大值为100，默认值为50
5. 在`VBoxContainer`下添加一个`Label`节点，用于显示当前亮度值

#### 2.3 音效设置

1. 在`VBoxContainer`下添加一个`VBoxContainer`节点，用于垂直布局音效设置相关组件
2. 在该`VBoxContainer`下添加一个`HBoxContainer`节点
3. 在`HBoxContainer`下添加一个`Label`节点，设置文本为"音效"
4. 在`HBoxContainer`下添加一个`CheckButton`节点，用于开关音效
5. 在`VBoxContainer`下添加另一个`HBoxContainer`节点
6. 在该`HBoxContainer`下添加一个`Label`节点，设置文本为"音效音量"
7. 在`HBoxContainer`下添加一个`HSlider`节点，设置最小值为0，最大值为100，默认值为75

#### 2.4 BGM设置

1. 在`VBoxContainer`下添加一个`VBoxContainer`节点，用于垂直布局BGM设置相关组件
2. 在该`VBoxContainer`下添加一个`HBoxContainer`节点
3. 在`HBoxContainer`下添加一个`Label`节点，设置文本为"BGM"
4. 在`HBoxContainer`下添加一个`CheckButton`节点，用于开关BGM
5. 在`VBoxContainer`下添加另一个`HBoxContainer`节点
6. 在该`HBoxContainer`下添加一个`Label`节点，设置文本为"BGM音量"
7. 在`HBoxContainer`下添加一个`HSlider`节点，设置最小值为0，最大值为100，默认值为70

#### 2.5 系统设置

1. 在`VBoxContainer`下添加一个`VBoxContainer`节点，用于垂直布局系统设置相关组件
2. 在该`VBoxContainer`下添加一个`HBoxContainer`节点
3. 在`HBoxContainer`下添加一个`Label`节点，设置文本为"系统音量"
4. 在`HBoxContainer`下添加一个`HSlider`节点，设置最小值为0，最大值为100，默认值为80
5. 在`VBoxContainer`下添加另一个`HBoxContainer`节点
6. 在该`HBoxContainer`下添加一个`Label`节点，设置文本为"全屏模式"
7. 在`HBoxContainer`下添加一个`CheckButton`节点，用于开关全屏模式

#### 2.6 按钮部分

1. 在`VBoxContainer`下添加一个`HBoxContainer`节点，用于水平布局按钮
2. 在`HBoxContainer`下添加一个`Button`节点，设置文本为"取消"
3. 在`HBoxContainer`下添加一个`Button`节点，设置文本为"保存"

### 3. 连接信号

1. 为`SettingsDialog`节点的`confirmed`信号连接到一个函数，用于处理保存设置的逻辑
2. 为`SettingsDialog`节点的`cancelled`信号连接到一个函数，用于处理取消设置的逻辑
3. 为每个`HSlider`节点的`value_changed`信号连接到相应的函数，用于更新显示的值
4. 为每个`CheckButton`节点的`toggled`信号连接到相应的函数，用于处理开关状态的变化

### 4. 创建设置脚本

1. 在`scripts/`目录下创建一个新的脚本文件，命名为`SettingsDialog.gd`
2. 脚本内容如下：

```gdscript
# 设置对话框脚本
extends ConfirmationDialog

# 导出变量，用于存储设置值
@export var brightness: float = 50.0
@export var sound_enabled: bool = true
@export var sound_volume: float = 75.0
@export var bgm_enabled: bool = true
@export var bgm_volume: float = 70.0
@export var system_volume: float = 80.0
@export var fullscreen: bool = false

# UI元素引用
@onready var brightness_slider = $VBoxContainer/VBoxContainer/HBoxContainer/HSlider
@onready var brightness_label = $VBoxContainer/VBoxContainer/Label
@onready var sound_check = $VBoxContainer/VBoxContainer2/HBoxContainer/CheckButton
@onready var sound_slider = $VBoxContainer/VBoxContainer2/HBoxContainer2/HSlider
@onready var bgm_check = $VBoxContainer/VBoxContainer3/HBoxContainer/CheckButton
@onready var bgm_slider = $VBoxContainer/VBoxContainer3/HBoxContainer2/HSlider
@onready var system_slider = $VBoxContainer/VBoxContainer4/HBoxContainer/HSlider
@onready var fullscreen_check = $VBoxContainer/VBoxContainer4/HBoxContainer2/CheckButton

# 当节点准备好时调用
func _ready() -> void:
	# 初始化UI元素的值
	brightness_slider.value = brightness
	brightness_label.text = str(int(brightness))
	sound_check.button_pressed = sound_enabled
	sound_slider.value = sound_volume
	bgm_check.button_pressed = bgm_enabled
	bgm_slider.value = bgm_volume
	system_slider.value = system_volume
	fullscreen_check.button_pressed = fullscreen

# 亮度值变化时调用
func _on_brightness_slider_value_changed(value: float) -> void:
	brightness = value
	brightness_label.text = str(int(value))

# 音效开关变化时调用
func _on_sound_check_toggled(button_pressed: bool) -> void:
	sound_enabled = button_pressed

# 音效音量变化时调用
func _on_sound_slider_value_changed(value: float) -> void:
	sound_volume = value

# BGM开关变化时调用
func _on_bgm_check_toggled(button_pressed: bool) -> void:
	bgm_enabled = button_pressed

# BGM音量变化时调用
func _on_bgm_slider_value_changed(value: float) -> void:
	bgm_volume = value

# 系统音量变化时调用
func _on_system_slider_value_changed(value: float) -> void:
	system_volume = value

# 全屏模式开关变化时调用
func _on_fullscreen_check_toggled(button_pressed: bool) -> void:
	fullscreen = button_pressed

# 保存设置
func save_settings() -> void:
	# 这里可以添加保存设置到配置文件的逻辑
	print("Settings saved")

# 取消设置
func cancel_settings() -> void:
	# 这里可以添加恢复默认设置的逻辑
	print("Settings cancelled")
```

### 5. 集成到暂停菜单

1. 在`PauseMenu.gd`脚本中添加对设置对话框的引用
2. 修改`_on_settings_button_pressed`函数，显示设置对话框

```gdscript
# 在PauseMenu.gd中添加
@onready var settings_dialog = preload("res://scenes/ui/SettingsDialog.tscn")

# 修改_on_settings_button_pressed函数
func _on_settings_button_pressed() -> void:
	var dialog = settings_dialog.instantiate()
	add_child(dialog)
	dialog.popup_centered()
```

| 滑块 | PascalCase | BrightnessSlider, SoundSlider |
| 复选按钮 | PascalCase | SoundCheck, FullscreenCheck |
| 按钮 | PascalCase | SaveButton, CancelButton |

## 注意事项

1. 确保所有UI组件的锚点和边距设置正确，以适应不同屏幕尺寸
2. 为设置项添加适当的默认值，确保游戏启动时设置合理
3. 考虑添加设置的持久化存储，使用Godot的`ConfigFile`类将设置保存到文件中
4. 为滑块添加适当的范围和步长，确保用户体验良好
5. 考虑添加设置的重置功能，允许用户恢复默认设置

## 总结

通过以上步骤，您可以在Godot引擎中创建一个功能完整的设置页面，包含亮度、音效、BGM和系统设置等选项。设置页面将通过暂停菜单中的设置按钮触发，允许玩家调整游戏的各种参数，提升游戏体验。
